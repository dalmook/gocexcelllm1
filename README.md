# gocexcelllm1

Tkinter 기반의 Excel + 메일 분석 앱입니다.

## 주요 기능

- 메일 분석
  - POP3 메일함 조회
  - 메일 선택 후 제목/본문 자동 채움
  - 구조화 분석
  - 핵심 요약
  - 자동 답장문구 생성
  - 표현 점검
  - 발표용 HTML 생성
- Excel 분석
  - 현재 Excel 선택영역 읽기
  - 요약 / 보고용 문구 / 주의점 분석

## 실행

```bash
python excel_selection_llm_app.py
```

## 메일 분석 빠른 사용법

1. 메일 분석 탭에서 사용자 ID와 비밀번호를 입력합니다.
2. `메일 조회`를 눌러 최근 메일 목록을 불러옵니다.
3. 목록에서 메일을 선택한 뒤 `선택 메일 불러오기`를 누릅니다.
4. 우측에서 구조화 분석, 핵심 요약, 자동 답장문구 생성을 실행합니다.

수동 입력도 계속 사용할 수 있습니다.

## Excel 분석 빠른 사용법

1. Excel 데스크톱에서 범위를 선택합니다.
2. 앱의 Excel 분석 탭에서 `선택영역 불러오기`를 누릅니다.
3. 필요하면 label/time/metric 옵션을 조정합니다.
4. `현재 설정으로 분석 실행`을 누릅니다.
